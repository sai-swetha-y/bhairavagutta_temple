# 🛕 Bhairavagutta Temple Dashboard

A web-based application to manage temple seva bookings and admin operations efficiently.

---

## 🚀 Features

### 👤 Devotee Booking

* Capture devotee details (Name, Phone, Gothram, etc.)
* Select seva / pooja offerings
* UPI payment via QR code
* UTR submission for payment tracking

### 📊 Admin Dashboard

* View all bookings in a structured table
* Group bookings by phone number
* Search by:

  * Name
  * Phone number
  * Seva
  * UTR
* Filter by:

  * Date range
  * Special days (Amavasya, Trayodashi)
  * Payment status
* Verify payments directly
* Export booking data (CSV)

### 📈 Analytics

* Total bookings count
* Total revenue
* Today’s bookings
* Verified vs Pending summary
* Revenue trend chart

### 💬 Communication

* WhatsApp notification after payment verification

---

## 🔐 Admin Access

* Password-protected login
* Session persistence (no repeated login)
* Logout functionality

---

## 🧰 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Google Apps Script
* **Database:** Google Sheets
* **Charts:** Chart.js

---

## 📂 Project Structure

* `index.html` → Devotee booking page
* `admin.html` → Admin dashboard
* `admin.js` → Dashboard logic
* `admin.css` → Styling
